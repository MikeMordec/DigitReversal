# DigitReversal

## Description

This C++ program includes a function, `reverseDigit`, which takes an integer as a parameter and returns the number with its digits reversed. For example, calling `reverseDigit(12345)` returns `54321`. Leading zeroes, if any, are not shown.

## Features

- **Digit Reversal**: Reverses the digits of the given integer.
- **Handles Negative Numbers**: Properly reverses digits while preserving the sign of negative numbers.
